# SmartRead — AI-Powered Multimodal eReading Platform

SmartRead is a web-based human-centred eReading platform for higher education that combines text, audio, game-based learning, and intelligent support tools in one accessible learning environment. It is designed to make reading-intensive study more engaging, inclusive, and easier to sustain.

## The Problem

To identify genuine user needs, I conducted a survey with university students to understand the challenges they face with academic reading and the features they would expect from a more effective eReading platform.

The findings highlighted recurring issues including low engagement, difficulty maintaining concentration, poor retention, and demand for more interactive, flexible, and accessible study tools.

These insights were translated directly into product decisions and led to the development of SmartRead as a practical solution for reading-intensive higher education study.

## Key Features

- Text reading mode
- Audio/text-to-speech mode
- Interactive quizzes
- Flashcards
- Dictionary support
- Notes & highlights
- XP / progress tracking
- Focus Mode
