# SmartRead Data Analysis & Product Insights

## Overview

This document summarises the data analysis conducted to understand student challenges with academic reading and identify features most valued in a modern eReading platform.

The findings informed the design and development of SmartRead.

This work demonstrates the ability to combine user research, analytics, and product development to solve a practical problem in higher education.

---

## Objective

The analysis aimed to answer three core questions:

1. What challenges do students face when reading academic materials?
2. How do students perceive technology-supported reading tools?
3. Which features should be prioritised in an improved eReading platform?

---

## Dataset Summary

- **178 student respondents**
- Responses collected through structured survey questions
- Participants from higher education backgrounds
- Responses analysed in aggregate form
- No personally identifiable data retained

---

## Analytical Approach

The dataset was analysed using standard data analytics methods, including:

- descriptive statistics
- percentage distribution analysis
- trend identification
- comparative interpretation
- feature-priority analysis

Where appropriate, Python / spreadsheet tools were used to organise and interpret results.

---

## Key Findings

---

## 1. Technology Increases Reading Engagement

Survey responses showed that most students believed computing tools improve engagement with reading materials.

<img width="975" height="483" alt="image" src="https://github.com/user-attachments/assets/8ef7787f-9be1-4fb6-b425-ed8ad026d118" />


### Insight

Students are receptive to digital solutions that make reading more interactive and easier to sustain.

### SmartRead Response

- progress tracking
- rewards system
- quizzes
- multimodal reading tools

---

## 2. Technology Supports Assimilation & Retention

A high proportion of respondents reported that digital tools help them understand and retain information more effectively.

<img width="975" height="483" alt="image" src="https://github.com/user-attachments/assets/85cad465-a74c-40ce-af82-195195f37001" />


### Insight

Students value tools that support active learning rather than passive reading alone.

### SmartRead Response

- flashcards
- comprehension quizzes
- concept matching
- notes & highlights

---

## 3. Students Want Better Features

Students prioritised stronger personalisation, interactivity, and integration in eReading platforms.

<img width="975" height="577" alt="image" src="https://github.com/user-attachments/assets/d34fbc46-5456-4a98-a7ba-ae41a7bf4759" />


### Insight

Students want modern platforms that adapt to their needs and combine multiple tools in one place.

### SmartRead Response

- text / audio / visual modes
- integrated dictionary
- intelligent assistant
- unified reading workspace

---

## 4. Common Reading Challenges

The survey highlighted recurring barriers including:

- difficulty maintaining concentration
- low motivation
- limited retention after reading
- unfamiliar terminology
- fragmented existing tools

### SmartRead Response

- Focus Mode
- streaks / XP motivation
- revision tools
- dictionary support
- all-in-one platform design

---

## Product Decision Framework

| User Insight | Product Decision |
|---|---|
| Students lose focus | Focus Mode |
| Students need flexible study methods | Text / Audio / Visual modes |
| Students need better retention tools | Flashcards / Quiz |
| Students need vocabulary help | Dictionary |
| Students value motivation | XP / Leaderboard |
| Students dislike fragmented tools | Unified SmartRead platform |
